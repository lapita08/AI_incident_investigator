# Security Policy

## Reporting A Vulnerability

Please do not open public issues for sensitive vulnerabilities. Email the maintainers listed in CODEOWNERS or use your organization's private disclosure process until a public security contact is configured.

## Supported Versions

This repository is pre-1.0. Security fixes are applied to `main`.

## Security Model

Version 1 is a local single-user MVP. It does not implement authentication, production authorization, or multi-tenancy. Do not expose it directly to the public internet.
